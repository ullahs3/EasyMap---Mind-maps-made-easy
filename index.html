Mind Map Controls

  • Bubble Creation & Editing
  • Right-click: Create bubble
  • Double-click: Edit text
  • Left-click drag: Move bubble
  • Scroll while dragging: Resize bubble
  • Connections
  • Right-click hold or Ctrl+Left-click: Connect bubbles
  • Left-click while connecting: Toggle line type
  • Deletion
  • Middle-click or Ctrl+Right-click: Delete bubble/line
