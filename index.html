Mind Map Controls

  • Bubble Creation & Editing
  • Right-click: Create bubble
  • Double-click: Edit text
  • Left-click: Move bubble
  • Left-click + Space: Move bubble and children
  • Left-click + Shift: Resize bubble
  • Shift + Right-click: Change bubble color
  • Connections
  • Ctrl + Left-click: Connect bubbles
  • Shift while connecting: Toggle line type
  • Deletion
  • Ctrl + Right-click: Delete bubble/line
